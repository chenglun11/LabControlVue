### Requirements
本项目基于Vue Admin
Vue.js+Element Plus

本项目前后端分离,且区分repo.
请务必通过 [后端仓库地址](https://github.com/chenglun11/LabControl) 打开


### Getting Started
1. Clone本项目并从WebStorm打开
```
# 测试
npm run dev
# 打包
npm run build
```

### Version Control
